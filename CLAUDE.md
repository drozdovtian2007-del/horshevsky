# План сайта-визитки — Horshevsky

## Общее
- Тип: одностраничный сайт (one-page, scroll)
- Навигация: боковой ползунок-индикатор (справа) с точками/линией для быстрого перехода между секциями
- Стиль: минимализм + благородный тёмно-синий акцент (#1a1a2e фон, #0f3460 / #4169E1 акцент, белый текст)
- Фишка: много 3D анимаций, нестандартные переходы, tech/code эстетика
- Стек: HTML + CSS + Vanilla JS + Three.js (для 3D) + GSAP (для анимаций)
- Хостинг: GitHub Pages (статика) или Railway

- Текст (на усмотрение разработчика, в стиле "молодой разработчик, специализируюсь на..."):

- Иконки стека: Python, JavaScript, HTML/CSS, Telegram API и др.
- Анимация: карточки вылетают при скролле (GSAP ScrollTrigger)

### 3. Портфолио
- Фильтры по категориям:
  - Telegram Боты
  - Сайты и лендинги
  - Telegram Mini Apps

### 4. Отзывы
- 6-8 отзывов от "клиентов" (placeholder имена и тексты)

  - "Сделал бота под мой бизнес за 3 дня, всё работает отлично!" — Алексей М.

  - и т.д.

### 5. Контакт / Связь
- Форма с полями:
  - Имя / Ник (текстовое поле)
  - Сообщение (textarea)
  - Кнопка: "Отправить сообщение"

  - Бот присылает в личку: имя, сообщение, IP/время (опционально)
- Анимация формы: 3D поворот карточки при фокусе на поле

## Боковой навигатор (справа)
- Вертикальная линия с точками для каждой секции
- Активная точка подсвечивается акцентным цветом
- Tooltip при наведении: название секции
- Плавный скролл при клике

---

## Анимации и 3D
- **Hero**: Three.js частицы / анимированная сетка (tech стиль)
- **Скролл**: GSAP ScrollTrigger — элементы вылетают, вращаются, появляются
- **Карточки портфолио**: CSS 3D tilt на hover (mouse parallax)
- **Переходы секций**: нестандартные (волна, разлёт частиц)
- **Курсор**: кастомный курсор с trail-эффектом
